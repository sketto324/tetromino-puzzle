# テトロミノパズル 🧩

これは、HTML, CSS, JavaScriptで作られたブラウザで遊べるテトロミノパズルゲームです。

## 遊び方

カラフルなブロックでパズルに挑戦！
10個のブロックを、下のボードにすきまなくピッタリはめよう。

-   **移動**: ブロックをドラッグ＆ドロップで動かせます。
-   **回転**: クリック（またはタップ）で回転します。
-   **反転**: PCでは**右クリック**、スマートフォンでは**ダブルタップ**で反転（裏返し）します。
-   **戻す**: 盤面に置いたブロックは、クリック（またはタップ）で元に戻せます。
